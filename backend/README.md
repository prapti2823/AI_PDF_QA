# AI PDF QA вЂ” Backend

A FastAPI backend that lets users upload a PDF and ask natural language questions about its content.
Powered by **LangChain**, **FAISS**, and **Hugging Face Transformers**.

---

```

LangChain splits text into overlapping chunks
```

## Quick Start

### 1. Create and activate a virtual environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python -m venv venv
source venv/bin/activate
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

> First run downloads the Hugging Face models (~90 MB). Subsequent runs use the local cache.

### 3. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env` вЂ” the defaults work out of the box with no API keys required.
To use OpenAI instead of the free Hugging Face model, add your key:

```
OPENAI_API_KEY=sk-...
```

### 4. Run the server

```bash
# From the backend/ directory
python -m app.main
```

Or with Uvicorn directly:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

The API will be available at **http://localhost:8000**

## Endpoints

### `GET /`
Health check.

**Response**
```json
{
  "status": "ok",
  "message": "AI PDF QA API is running."
}
```

---

### `POST /upload`
Upload a PDF file. Returns a `session_id` used for all subsequent questions.

**Request** вЂ” `multipart/form-data`

| Field | Type | Description        |
|-------|------|--------------------|
| file  | File | PDF file to upload |

**Response `200`**
```json
{
  "message": "PDF uploaded and processed successfully.",
  "session_id": "3f7a1c2e-...",
  "filename": "report.pdf",
  "pages": 12,
  "chunks": 47
}
```

**Error responses**

| Code | Reason                              |
|------|-------------------------------------|
| 400  | File is not a PDF                   |
| 413  | File exceeds size limit             |
| 422  | PDF is empty, scanned, or corrupted |
| 500  | Unexpected server error             |

---

### `POST /ask`
Ask a question about the uploaded PDF.

**Request body** вЂ” `application/json`
```json
{
  "question": "What is the main conclusion of the report?",
  "session_id": "3f7a1c2e-..."
}
```

**Response `200`**
```json
{
  "question": "What is the main conclusion of the report?",
  "answer": "The main conclusion is that renewable energy adoption...",
  "session_id": "3f7a1c2e-..."
}
```

**Error responses**

| Code | Reason                                  |
|------|-----------------------------------------|
| 404  | session_id not found (upload PDF first) |
| 500  | Error generating answer                 |

## Environment Variables

| Variable           | Default                                      | Description                              |
|--------------------|----------------------------------------------|------------------------------------------|
| `OPENAI_API_KEY`   | _(empty)_                                    | Optional. Uses HuggingFace if not set.   |
| `HUGGINGFACE_API_KEY` | _(empty)_                                 | Optional. For private/gated HF models.  |
| `EMBEDDING_MODEL`  | `sentence-transformers/all-MiniLM-L6-v2`     | Sentence Transformer model name          |
| `QA_MODEL`         | `deepset/roberta-base-squad2`                | Extractive QA model name                 |
| `CHUNK_SIZE`       | `500`                                        | Max characters per text chunk            |
| `CHUNK_OVERLAP`    | `50`                                         | Overlap between consecutive chunks       |
| `UPLOAD_DIR`       | `uploads`                                    | Temporary file storage directory         |
| `MAX_FILE_SIZE_MB` | `20`                                         | Maximum PDF upload size in MB            |
| `HOST`             | `0.0.0.0`                                    | Server bind address                      |
| `PORT`             | `8000`                                       | Server port                              |

## Limitations

- Vector stores are kept **in memory** вЂ” restarting the server clears all sessions.
- Scanned PDFs (image-only) are not supported вЂ” text extraction requires selectable text.
- The free HuggingFace QA model is **extractive** (finds spans in the text) rather than generative.
